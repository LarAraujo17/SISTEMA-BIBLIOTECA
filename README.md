📚 Sistema de Biblioteca
Este projeto foi desenvolvido como parte da disciplina Programação Web do curso de Análise e Desenvolvimento de Sistemas.
O objetivo é implementar um sistema de gerenciamento de biblioteca, permitindo o cadastro e controle de usuários, livros e empréstimos.

⚙️ Tecnologias utilizadas
- Node.js com Express para criação do servidor
- CORS para integração entre front-end e back-end
- MySQL para persistência dos dados
- phpMyAdmin para administração do banco
- Estrutura modular com rotas dedicadas para usuários, livros e empréstimos

🚀 Funcionalidades principais
- Cadastro e gerenciamento de usuários
- Registro e controle de livros
- Sistema de empréstimos com integração ao banco de dados
- Interface web para interação com o sistema
